# ATENCIÓN: Eliminación de SQLite para Railway

- Se han eliminado las referencias a init-db.js y better-sqlite3.
- El proyecto ahora solo usará PostgreSQL (Railway).
- Si tienes lógica de inicialización de tablas, deberás migrarla a scripts compatibles con PostgreSQL.
- Si necesitas migrar datos de SQLite a PostgreSQL, avísame y te ayudo.

¡Ya puedes intentar desplegar de nuevo en Railway!
